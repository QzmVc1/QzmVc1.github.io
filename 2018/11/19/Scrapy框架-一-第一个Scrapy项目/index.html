1.items.py

дҪҝз”ЁitemsжқҘеҢ…иЈ…йңҖиҰҒзҲ¬еҸ–зҡ„еҶ…е®№гҖӮжҠҠеҶ…е®№йғҪз”Ёitems.pyжқҘиҝӣиЎҢз®ЎзҗҶпјҢдҫҝдәҺжҠҠжҠ“еҸ–зҡ„еҶ…е®№дј йҖ’иҝӣpipelinesиҝӣиЎҢеҗҺжңҹеӨ„зҗҶгҖӮеҗҢж—¶пјҢжҠҠеҶ…е®№йғҪж”ҫиҝӣitems.pyд»ҘеҗҺпјҢеҸҜд»Ҙи§ЈиҖҰеҗҲзҲ¬иҷ«ж–Үд»¶demo.pyпјҢиҙЈд»»жӣҙеҠ жҳҺжҷ°пјҡзҲ¬иҷ«иҙҹиҙЈеҺ»еҸ‘иҜ·жұӮпјҢи§ЈжһҗзҪ‘еқҖпјӣitems.pyиҙҹиҙЈз®ЎзҗҶжҠ“еҸ–еҲ°зҡ„еҶ…е®№гҖӮ

еңЁйЎ№зӣ®зӣ®еҪ•дёӢжңүitems.pyж–Үд»¶гҖӮиҝҷжҳҜеӯҳж”ҫitemsзҡ„ең°ж–№пјҢд№ҹе°ұжҳҜеӯҳж”ҫжҠ“еҸ–еҶ…е®№зҡ„ең°ж–№гҖӮжҲ‘д»¬йңҖиҰҒеңЁitems.pyдёӯе‘ҠиҜүScrapyжҲ‘д»¬иҰҒжҠ“еҸ–зҡ„еҶ…е®№еҸ«д»Җд№ҲеҗҚеӯ—пјҢд№ҹе°ұжҳҜйңҖиҰҒеЈ°жҳҺitemsгҖӮ

ItemдҪҝз”Ёз®ҖеҚ•зҡ„classе®ҡд№үиҜӯжі•д»ҘеҸҠ Field еҜ№иұЎжқҘеЈ°жҳҺгҖӮдҫӢеҰӮ:

иЎЁзӨәжҲ‘д»¬иҰҒжҠ“еҸ–зҡ„еҶ…е®№жҳҜпјҡtitleпјҢtimeпјҢauthorд»ҘеҸҠlinkгҖӮ

FieldеҜ№иұЎжҢҮжҳҺдәҶжҜҸдёӘеӯ—ж®өзҡ„е…ғж•°жҚ®(metadata)пјҢдҪ еҸҜд»ҘдёәжҜҸдёӘеӯ—ж®өжҢҮжҳҺд»»дҪ•зұ»еһӢзҡ„е…ғж•°жҚ®гҖӮ FieldеҜ№иұЎеҜ№жҺҘеҸ—зҡ„еҖјжІЎжңүд»»дҪ•йҷҗеҲ¶гҖӮд№ҹжӯЈжҳҜеӣ дёәиҝҷдёӘеҺҹеӣ пјҢж–ҮжЎЈд№ҹж— жі•жҸҗдҫӣжүҖжңүеҸҜз”Ёзҡ„е…ғж•°жҚ®зҡ„й”®(key)еҸӮиҖғеҲ—иЎЁгҖӮ FieldеҜ№иұЎдёӯдҝқеӯҳзҡ„жҜҸдёӘй”®еҸҜд»Ҙз”ұеӨҡдёӘз»„д»¶дҪҝз”ЁпјҢе№¶дё”еҸӘжңүиҝҷдәӣз»„д»¶зҹҘйҒ“иҝҷдёӘй”®зҡ„еӯҳеңЁгҖӮдҪ еҸҜд»Ҙж №жҚ®иҮӘе·ұзҡ„йңҖжұӮпјҢе®ҡд№үдҪҝз”Ёе…¶д»–зҡ„Fieldй”®гҖӮ и®ҫзҪ® Field еҜ№иұЎзҡ„дё»иҰҒзӣ®зҡ„е°ұжҳҜеңЁдёҖдёӘең°ж–№е®ҡд№үеҘҪжүҖжңүзҡ„е…ғж•°жҚ®гҖӮ дёҖиҲ¬жқҘиҜҙпјҢйӮЈдәӣдҫқиө–жҹҗдёӘеӯ—ж®өзҡ„з»„д»¶иӮҜе®ҡдҪҝз”ЁдәҶзү№е®ҡзҡ„й”®(key)гҖӮжӮЁеҝ…йЎ»жҹҘзңӢз»„д»¶зӣёе…ізҡ„ж–ҮжЎЈпјҢжҹҘзңӢе…¶з”ЁдәҶе“Әдәӣе…ғж•°жҚ®й”®(metadata key)гҖӮ

йңҖиҰҒжіЁж„Ҹзҡ„жҳҜпјҢз”ЁжқҘеЈ°жҳҺitemзҡ„FieldеҜ№иұЎе№¶жІЎжңүиў«иөӢеҖјдёәclassзҡ„еұһжҖ§гҖӮ дёҚиҝҮжӮЁеҸҜд»ҘйҖҡиҝҮItem.fieldsеұһжҖ§иҝӣиЎҢи®ҝй—®гҖӮ

2.demo.py


жҺҘдёӢжқҘжҲ‘д»¬иҝӣе…Ҙdemo.py ж–Үд»¶дёӯгҖӮimportиҜӯеҸҘиғҪеӨҹи®©жҲ‘д»¬дҪҝз”ЁScrapyжЎҶжһ¶дёӯе·Іжңүзҡ„зұ»гҖӮдёӢйқўжҳҜжү©еұ•иҮӘscrapy.Spiderзҡ„DemoSpiderзұ»зҡ„е®ҡд№үгҖӮйҖҡиҝҮжү©еұ•зҡ„ж–№ејҸпјҢе°Ҫз®ЎжҲ‘д»¬е®һйҷ…дёҠжІЎжңүеҶҷд»»дҪ•д»Јз ҒпјҢдҪҶжҳҜиҜҘзұ»е·Із»Ҹз»§жүҝдәҶScrapyжЎҶжһ¶дёӯSpiderзұ»зҡ„зӣёеҪ“дёҖйғЁеҲҶеҠҹиғҪгҖӮиҝҷж ·пјҢе°ұеҸҜд»ҘеҸӘйўқеӨ–зј–еҶҷе°‘йҮҸзҡ„д»Јз ҒиЎҢпјҢиҖҢиҺ·еҫ—дёҖдёӘе®Ңж•ҙиҝҗиЎҢзҡ„зҲ¬иҷ«дәҶгҖӮжңҖеҗҺжҳҜз©әеҮҪж•°parse()зҡ„е®ҡд№үпјҢиҜҘеҮҪж•°еҢ…еҗ«дёӨдёӘеҸӮж•°selfе’ҢresponseеҜ№иұЎгҖӮйҖҡиҝҮselfзҡ„еј•з”ЁпјҢжҲ‘д»¬е°ұеҸҜд»ҘдҪҝз”ЁзҲ¬иҷ«дёӯж„ҹе…ҙи¶Јзҡ„еҠҹиғҪдәҶгҖӮresponseеҜ№иұЎжҳҜзҪ‘йЎөиҝ”еӣһзҡ„иҜ·жұӮеҜ№иұЎгҖӮ

  • йҰ–е…ҲпјҢжҲ‘д»¬йңҖиҰҒеј•е…Ҙд№ӢеүҚеңЁitem.pyдёӯзј–еҶҷзҡ„DemoItemзұ»гҖӮеҰӮеүҚжүҖиҝ°пјҢе®ғеңЁDemoзӣ®еҪ•зҡ„items.pyж–Үд»¶дёӯпјҢд№ҹе°ұжҳҜDemo.itemsжЁЎеқ—дёӯгҖӮжҲ‘д»¬дҪҝз”ЁеҰӮдёӢе‘Ҫд»Өеј•е…ҘиҜҘжЁЎеқ—пјҡ

    from Demo.items import DemoItem

  • з„¶еҗҺйңҖиҰҒиҝӣиЎҢе®һдҫӢеҢ–пјҢе№¶иҝ”еӣһдёҖдёӘеҜ№иұЎгҖӮиҝҷйқһеёёз®ҖеҚ•пјҢеңЁparse()ж–№жі•дёӯпјҢеҸҜд»ҘйҖҡиҝҮж·»еҠ  item = DemoItem() иҜӯеҸҘеҲӣе»әдёҖдёӘж–°зҡ„itemпјҢз„¶еҗҺеҸҜд»ҘжҢүеҰӮдёӢж–№ејҸдёәе…¶еӯ—ж®өеҲҶй…ҚиЎЁиҫҫејҸпјҡ

    item[вҖҳtitleвҖҷ] = response.xpath(вҖҳ вҖҳ).extract()

дёүгҖҒжё…зҗҶвҖ”itemиЈ…иҪҪеҷЁдёҺз®ЎзҗҶеӯ—ж®ө

йҰ–е…ҲпјҢжҲ‘д»¬дҪҝз”ЁдёҖдёӘејәеӨ§зҡ„е·Ҙе…·зұ»вҖ”вҖ”ItemLoaderпјҢд»Ҙд»ЈжӣҝйӮЈдәӣжқӮд№ұзҡ„extract()е’Ңxpath()ж“ҚдҪңгҖӮйҖҡиҝҮдҪҝз”ЁиҜҘзұ»пјҢжҲ‘д»¬зҡ„parse()ж–№жі•дјҡжҢүеҰӮдёӢиҝӣиЎҢд»Јз ҒеҸҳжӣҙпјҡ

еҘҪеӨҡдәҶпјҢжҳҜдёҚжҳҜпјҹдёҚиҝҮпјҢиҝҷз§ҚеҶҷжі•е№¶дёҚеҸӘжҳҜи§Ҷи§үдёҠжӣҙеҠ иҲ’йҖӮпјҢе®ғиҝҳйқһеёёжҳҺзЎ®ең°еЈ°жҳҺдәҶжҲ‘д»¬ж„ҸеӣҫеҺ»еҒҡзҡ„дәӢжғ…пјҢиҖҢдёҚдјҡе°Ҷе…¶дёҺе®һзҺ°з»ҶиҠӮж··ж·Ҷиө·жқҘгҖӮиҝҷе°ұдҪҝеҫ—д»Јз Ғе…·жңүжӣҙеҘҪзҡ„еҸҜз»ҙжҠӨжҖ§д»ҘеҸҠиҮӘжҸҸиҝ°жҖ§гҖӮ

ItemLoaderжҸҗдҫӣдәҶи®ёеӨҡжңүи¶Јзҡ„з»“еҗҲж•°жҚ®еҸҠеҜ№ж•°жҚ®иҝӣиЎҢж јејҸеҢ–е’Ңжё…жҙ—зҡ„ж–№ејҸгҖӮеӨ„зҗҶеҷЁжҳҜдёҖдёӘеҝ«йҖҹиҖҢеҸҲз®ҖеҚ•зҡ„еҮҪж•°гҖӮеӨ„зҗҶеҷЁзҡ„дёҖдёӘдҫӢеӯҗжҳҜJoin()гҖӮеҒҮи®ҫдҪ е·Із»ҸдҪҝз”Ёзұ»дјј//p зҡ„xpathиЎЁиҫҫејҸйҖүеҸ–дәҶеҫҲеӨҡдёӘж®өиҗҪпјҢиҜҘеӨ„зҗҶеҷЁеҸҜд»Ҙе°Ҷиҝҷдәӣж®өиҗҪз»“еҗҲжҲҗдёҖдёӘжқЎзӣ®гҖӮеҸҰдёҖдёӘйқһеёёжңүж„ҸжҖқзҡ„еӨ„зҗҶеҷЁжҳҜMapCompose(),йҖҡиҝҮдҪҝз”ЁиҜҘеӨ„зҗҶеҷЁпјҢдҪ еҸҜд»ҘдҪҝз”Ёд»»ж„ҸPythonеҮҪж•°жҲ–PythonеҮҪж•°й“ҫпјҢд»Ҙе®һзҺ°еӨҚжқӮзҡ„еҠҹиғҪгҖӮжҜ”еҰӮпјҢMapCompose(float)еҸҜд»Ҙе°Ҷеӯ—з¬ҰдёІж•°жҚ®иҪ¬жҚўдёәж•°еҖјпјҢиҖҢMapCompose(str.strip, str.title)еҸҜд»ҘеҲ йҷӨеӨҡдҪҷзҡ„з©әзҷҪз¬ҰпјҢе№¶е°Ҷеӯ—з¬ҰдёІж јејҸеҢ–дёәжҜҸдёӘеҚ•иҜҚеқҮдёәйҰ–еӯ—жҜҚеӨ§еҶҷзҡ„ж ·ејҸгҖӮеҪ“з„¶пјҢдҪ д№ҹеҸҜд»ҘйҖҡиҝҮlambdaиЎЁиҫҫејҸиҮӘе®ҡд№үдёҖдәӣеҮҪж•°гҖӮ

иҝҷйҮҢиҰҒи§ЈеҶізҡ„е…ій”®й—®йўҳжҳҜпјҢеӨ„зҗҶеҷЁеҸӘжҳҜдёҖдәӣз®ҖеҚ•е°Ҹе·§зҡ„еҠҹиғҪпјҢз”ЁжқҘеҜ№жҲ‘д»¬зҡ„Xpath/CSSз»“жһңиҝӣиЎҢеҗҺзҪ®еӨ„зҗҶгҖӮзҺ°еңЁпјҢеңЁзҲ¬иҷ«дёӯдҪҝз”ЁеҮ дёӘиҝҷж ·зҡ„еӨ„зҗҶеҷЁпјҢе№¶жҢүз…§жҲ‘д»¬жғіиҰҒзҡ„ж–№ејҸиҫ“еҮәпјҢд»Јз ҒеҰӮдёӢпјҡ

жңҖеҗҺпјҢжҲ‘д»¬еҸҜд»ҘйҖҡиҝҮдҪҝз”Ёadd_value()ж–№жі•пјҢж·»еҠ Pythonи®Ўз®—еҫ—еҮәзҡ„еҚ•дёӘеҖјпјҢжҲ‘д»¬еҸҜд»Ҙз”ЁиҜҘж–№жі•и®ҫзҪ®вҖңз®ЎзҗҶеӯ—ж®өвҖқпјҢжҜ”еҰӮURLпјҢзҲ¬иҷ«еҗҚз§°пјҢж—¶й—ҙжҲізӯүгҖӮжҲ‘д»¬иҝҳеҸҜд»ҘзӣҙжҺҘдҪҝз”Ёз®ЎзҗҶеӯ—ж®өиЎЁдёӯжҖ»з»“еҮәжқҘзҡ„иЎЁиҫҫејҸпјҡ

д»ҘдёҠжҳҜItemLoaderеҸҠе…¶еҠҹиғҪзҡ„з®ҖиҰҒжҰӮиҝ°пјҢItemLoaderе’ҢеӨ„зҗҶеҷЁжҳҜеҹәдәҺзј–еҶҷе№¶ж”ҜжҢҒдәҶжҲҗеҚғдёҠдёҮдёӘзҲ¬иҷ«зҡ„дәәд»¬зҡ„жҠ“еҸ–йңҖжұӮиҖҢејҖеҸ‘зҡ„е·Ҙе…·еҢ…пјҢеҰӮжһңдҪ еҮҶеӨҮејҖеҸ‘еӨҡдёӘзҲ¬иҷ«зҡ„иҜқпјҢе°ұйқһеёёеҖјеҫ—еҺ»еӯҰд№ дҪҝз”Ёе®ғд»¬гҖӮ

еӣӣгҖҒScrapy Shell

Scrapyз»Ҳз«ҜжҳҜдёҖдёӘдәӨдә’з»Ҳз«ҜпјҢжҲ‘д»¬еҸҜд»ҘеңЁжңӘеҗҜеҠЁspiderзҡ„жғ…еҶөдёӢе°қиҜ•еҸҠи°ғиҜ•д»Јз ҒпјҢд№ҹеҸҜд»Ҙз”ЁжқҘжөӢиҜ•XPathжҲ–CSSиЎЁиҫҫејҸпјҢжҹҘзңӢд»–д»¬зҡ„е·ҘдҪңж–№ејҸпјҢж–№дҫҝжҲ‘д»¬зҲ¬еҸ–зҡ„зҪ‘йЎөдёӯжҸҗеҸ–зҡ„ж•°жҚ®гҖӮ

еҰӮжһңе®үиЈ…дәҶ IPython пјҢScrapyз»Ҳз«Ҝе°ҶдҪҝз”Ё IPython (жӣҝд»Јж ҮеҮҶPythonз»Ҳз«Ҝ)гҖӮ IPython з»Ҳз«ҜдёҺе…¶д»–зӣёжҜ”жӣҙдёәејәеӨ§пјҢжҸҗдҫӣжҷәиғҪзҡ„иҮӘеҠЁиЎҘе…ЁпјҢй«ҳдә®иҫ“еҮәпјҢеҸҠе…¶д»–зү№жҖ§гҖӮпјҲжҺЁиҚҗе®үиЈ…IPythonпјү

4.1 еҗҜеҠЁScrapy Shell

иҝӣе…ҘcmdпјҢжү§иЎҢдёӢеҲ—е‘Ҫд»ӨжқҘеҗҜеҠЁshell:

scrapy shell вҖңhttp://jcoffeezph.top"

дјҡеңЁжҺ§еҲ¶еҸ°зңӢеҲ°еҰӮдёӢиҫ“еҮәпјҡ

Scrapy Shellж №жҚ®дёӢиҪҪзҡ„йЎөйқўдјҡиҮӘеҠЁеҲӣе»әдёҖдәӣж–№дҫҝдҪҝз”Ёзҡ„еҜ№иұЎпјҢдҫӢеҰӮ Response еҜ№иұЎпјҢд»ҘеҸҠ Selector еҜ№иұЎ (еҜ№HTMLеҸҠXMLеҶ…е®№)гҖӮ

  • еҪ“shellиҪҪе…ҘеҗҺпјҢе°Ҷеҫ—еҲ°дёҖдёӘеҢ…еҗ«responseж•°жҚ®зҡ„жң¬ең° response еҸҳйҮҸпјҢиҫ“е…Ҙresponse.bodyе°Ҷиҫ“еҮәresponseзҡ„еҢ…дҪ“пјҢиҫ“еҮә response.headers еҸҜд»ҘзңӢеҲ°responseзҡ„еҢ…еӨҙгҖӮ
  • иҫ“е…Ҙ response.selector ж—¶пјҢ е°ҶиҺ·еҸ–еҲ°дёҖдёӘresponse еҲқе§ӢеҢ–зҡ„зұ» Selector зҡ„еҜ№иұЎпјҢжӯӨж—¶еҸҜд»ҘйҖҡиҝҮдҪҝз”Ёresponse.selector.xpath()жҲ–response.selector.css() жқҘеҜ№response иҝӣиЎҢжҹҘиҜўгҖӮ
  • Scrapyд№ҹжҸҗдҫӣдәҶдёҖдәӣеҝ«жҚ·ж–№ејҸ, дҫӢеҰӮ response.xpath()жҲ–response.css()еҗҢж ·еҸҜд»Ҙз”ҹж•ҲгҖӮ

4.2 SelectorsйҖүжӢ©еҷЁ

Scrapy Selectors еҶ…зҪ® XPath е’Ң CSS Selector иЎЁиҫҫејҸжңәеҲ¶

SelectorжңүеӣӣдёӘеҹәжң¬зҡ„ж–№жі•пјҢжңҖеёёз”Ёзҡ„иҝҳжҳҜxpath:

  • xpath(): дј е…ҘxpathиЎЁиҫҫејҸпјҢиҝ”еӣһиҜҘиЎЁиҫҫејҸжүҖеҜ№еә”зҡ„жүҖжңүиҠӮзӮ№зҡ„selector listеҲ—иЎЁ
  • extract(): еәҸеҲ—еҢ–иҜҘиҠӮзӮ№дёәUnicodeеӯ—з¬ҰдёІе№¶иҝ”еӣһlist
  • css(): дј е…ҘCSSиЎЁиҫҫејҸпјҢиҝ”еӣһиҜҘиЎЁиҫҫејҸжүҖеҜ№еә”зҡ„жүҖжңүиҠӮзӮ№зҡ„selector listеҲ—иЎЁпјҢиҜӯжі•еҗҢBeautifulSoup4
  • re(): ж №жҚ®дј е…Ҙзҡ„жӯЈеҲҷиЎЁиҫҫејҸеҜ№ж•°жҚ®иҝӣиЎҢжҸҗеҸ–пјҢиҝ”еӣһUnicodeеӯ—з¬ҰдёІlistеҲ—иЎЁ

4.3 XPathиЎЁиҫҫејҸзҡ„дҫӢеӯҗеҸҠеҜ№еә”зҡ„еҗ«д№ү:

жҲ‘д»¬иҝҳжҳҜд»Ҙжј”зӨәеҚҡе®ў https://jcoffeezph.top/ дёәдҫӢпјҡ

дёӢйқўеҜ№4.3.1еҒҡеҮ зӮ№и§ЈйҮҠпјҡ

  1. xpathиЎЁиҫҫејҸиҜӯжі•дёҚеҒҡиҝҮеӨҡзҡ„иҜҙжҳҺпјҢе…·дҪ“еҸҜд»ҘеҸӮиҖғжҲ‘зҡ„дёҖзҜҮеҚҡе®ўпјҡXpathз”Ёжі•жҖ»з»“

  2. еҰӮжһңдёҚдҪҝз”Ёextract()пјҢresponse.xpath()зҡ„иҝ”еӣһеҖјжҳҜзҪ‘йЎөеҶ…е®№йў„еҠ иҪҪзҡ„SelectorеҜ№иұЎпјҢдёәдәҶиҺ·еҸ–зңҹе®һеҖјпјҢеҸҜд»ҘдҪҝз”Ёextract()пјҢд№ҹеҸҜд»ҘдҪҝз”Ёre()гҖӮ

  3. extract()еҗҚдёәжҸҗеҸ–пјҢиҝ”еӣһдёҖдёӘlistеҲ—иЎЁпјҢйҮҢйқўжҳҜдёҖдәӣиҰҒжҸҗеҸ–зҡ„еҶ…е®№гҖӮиҝҷйҮҢжңүдёӘеқ‘пјҢеңЁshellдёӯиҫ“е…Ҙextract()[0]и®ҝй—®з¬¬дёҖдёӘеҲ—иЎЁе…ғзҙ жІЎжңүй—®йўҳпјҢдҪҶеңЁpycharmдёӯеҚҙжҠҘindex out of rangeиҝҷдёӘй”ҷпјҢиЎЁзӨәеҫҲеҘҮжҖӘпјҢзҺ°еңЁд№ҹдёҚзҹҘйҒ“е…·дҪ“еҺҹеӣ пјҢpycharmдёӯе…·дҪ“и§ЈеҶіж–№жі•еҸҜд»ҘеҸӮиҖғжҲ‘зҡ„дёҖзҜҮеҚҡе®ўпјҡScrapyжЎҶжһ¶д№Ӣз»Ҷж•°еҲ°зӣ®еүҚдёәжӯўжҲ‘йҒҮеҲ°зҡ„еқ‘

д»ҘеҗҺеҒҡж•°жҚ®жҸҗеҸ–зҡ„ж—¶еҖҷпјҢеҸҜд»Ҙе…ҲеңЁScrapy ShellдёӯжөӢиҜ•пјҢжөӢиҜ•йҖҡиҝҮеҗҺеҶҚеә”з”ЁеҲ°д»Јз ҒдёӯгҖӮиҖҢScrapy Shellзҡ„дё»иҰҒдҪңз”Ёд№ҹжҳҜз”ЁжқҘжөӢиҜ•xpathиЎЁиҫҫејҸгҖӮ

еҪ“з„¶Scrapy ShellдҪңз”ЁдёҚд»…д»…еҰӮжӯӨпјҢдҪҶжҳҜдёҚеұһдәҺжҲ‘д»¬иҜҫзЁӢйҮҚзӮ№пјҢдёҚеҒҡиҜҰз»Ҷд»Ӣз»ҚгҖӮ
